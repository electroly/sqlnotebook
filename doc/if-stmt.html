IF Statement

Takes one code path or the other based on an SQL expression. The condition expression must evaluate to the integer 0 or 1.

Syntax


Arguments

  • condition (integer, 0 or 1)
    If this expression evaluates to 1, then the stmt-if-true statements are executed.  Otherwise, the stmt-if-false statements are executed.
  • stmt-if-true (statement)
    The statements to execute if the condition is 1.  If more than one stmt-if-true statement is desired, the BEGIN and END keywords must be used.
  • stmt-if-false (statement)
    The statements to execute if the condition is 0.  If more than one stmt-if-false statement is desired, the BEGIN and END keywords must be used.

Examples

  • IF 1 PRINT 'Hi';
    Prints "Hi".
  • IF 0 PRINT 'Hi';
    Prints nothing.
  • IF 1 + 1 = 2 PRINT 'Foo' ELSE PRINT 'Bar';
    Prints "Foo".
  • IF 2 > 3
        PRINT 'Foo'
    ELSE
        BEGIN
        PRINT 'Bar';
        PRINT 'Baz';
        END
    Prints "Bar" and "Baz".
  • IF @x = 'foo' PRINT 'x is foo'
    ELSE IF @x = 'bar' PRINT 'x is bar'
    ELSE PRINT 'x is neither foo nor bar'
    Prints whether the variable @x is "foo", "bar", or neither.
