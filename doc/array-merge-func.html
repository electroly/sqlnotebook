ARRAY_MERGE Function

Creates a new array by merging two or more arrays, in order.

Syntax

ARRAY_MERGE(array1, array2, [array3], ...)

Parameters

  • array1, array2, array3, ...: blob
    The arrays to merge together.

Return Value

A new array formed by merging the arrays together.

Example

DECLARE @a = ARRAY(111, 222);
DECLARE @b = ARRAY(333, 444);
DECLARE @c = ARRAY_MERGE(@a, @b);
PRINT @c;  -- "[111, 222, 333, 444]"
